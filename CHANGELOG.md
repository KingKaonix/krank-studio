# Changelog

## v1.1.0
- Renamed to KRANK STUDIO
- 8 guitar effects (Distortion, Amp Sim, EQ, Chorus, Noise Gate, Compressor, Delay, Reverb)
- Chromatic tuner with 12 tuning presets
- Tone matcher with spectral analysis and effect recommendations
- Song-to-tablature transcription with polyphonic mode
- Metronome with tap tempo (40-240 BPM)
- Looper pedal with record/play/overdub/clear
- MIDI footswitch support with learn mode and looper toggle
- BLE foot controller scanning UI
- Preset save/load from files
- Audio input monitoring via headphones
- Tuner mute (dry signal cut)
- Real-time VU meter
- WAV/MP3/AAC/OGG file loading + mic recording + device audio capture
- Low-latency Oboe audio engine
